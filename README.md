# Beach Landing - WW2 Bunker Defense Shooter

A high-immersion WW2 beach defense game built with HTML5 Canvas and Web Audio API.

## How to Play

**Desktop:**
- Mouse movement to aim
- Left-click or hold to shoot
- Crosshair in center of screen

**Mobile:**
- Tap to aim and auto-fire
- Touch anywhere on screen to shoot continuously

## Game Features

- **Progressive Difficulty:** 5 waves with increasing enemy complexity and fire rate
- **5 Enemy Types:** Soldiers, Elites, Grenadiers, Tanks, and Planes
- **Part-Based Damage System:** Destroy enemy weak points (ammo, fuel tanks, engines)
- **5 Tier Weapon System:** Unlock stronger weapons as you get more kills
- **Realistic Audio:** Procedural sound effects (no external audio files)
- **Responsive Design:** Full support for desktop and mobile devices

## Deployment

Hosted on Vercel. Play online or open `beach_landing.html` in any modern web browser.
